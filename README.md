### InContext learning

1. Poser la question à un modèle de langage
2. Ajouter de la connaissance : Fiche déclaration de créance https://drive.google.com/file/d/1ZyZL9uajCLDJGPtoAPEyfPWwPYgqbItr/view?usp=drive_link
